    Construção de um modelo para Previsão de Vendas de uma Rede de Farmácias

    Neste projeto foram utilizados conceitos e modelos de Machine Learning - como XGBoost e Random Forest - para realizar a previsão de vendas das próximas 6 semanas da rede de farmácias Rossmann para que o CFO da empresa possa saber o quanto investir em reformas das filiais.

    O resultado final foi a criação de um Bot no Telegram, hospedado em um ambiente Cloud e conectado através de uma API. O Bot pode ser acessado por qualquer dispositivo com Telegram conectado à internet para obtenção das informações de previsão de vendas.

    • Python, Pandas, Numpy e Scikit Learn.

    • Machine Learning com XGBoost Regressor, Random Forest Regressor e Linear Regressor.
